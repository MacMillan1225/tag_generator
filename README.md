## Q&A
**Q**：怎么用？

**A**：运行脚本，输入需要的tag，直到输入"s"字符为止，输出一个包含了所有图片的tag文件夹和一个tag图片

&nbsp;

**Q**：自定义？

**A**：打开main.py，前五行都可以改，代表了颜色值、横向长度（像素）、横向间隔（像素）、纵向间隔（像素）、单个标签路径

&nbsp;

**Q**：这么粗糙就传github？
